Node Cheat Sheet

  1. Location of script short cuts : packages.json

    "scripts": { "start": "node index.js" }

  2. Start node server : npm start

  3. Check node version : node -v

  4. Update packages : npm update

  5. Setup debuggung npm start => "@powershell $env:DEBUG='*,-express:router*' ;node index.js"
